2. **引导Agent阅读指令**
   ```

   ```

3. **启动第一个实验**
   ```
   根据program.md的指导，开始第一个改进实验。

   3. 进行修改
   4. 运行 python train.py 验证效果
   5. 报告结果并决定是否保留改动
   ```

## 💡 Agent可以探索的改进方向

### 🔥 简单快速（推荐先尝试）
1. **调整学习率** - `5e-4, 1e-4, 5e-5`
2. **调整dropout** - `0.2, 0.3, 0.4`
3. **调整权重衰减** - `1e-4, 1e-5`

### 🎯 中等难度
4. **增加GNN层数** - `4层, 5层`
5. **调整隐藏维度** - `128, 256`
6. **改变优化器** - `Adam, SGD+momentum`

### 🚀 高级探索
7. **添加Batch Normalization**
8. **尝试不同的激活函数** - `LeakyReLU, GELU`
9. **调整梯度累积步数**

## ⚠️ 重要提醒

4. **失败也要分析** - 理解为什么某些改动不行
