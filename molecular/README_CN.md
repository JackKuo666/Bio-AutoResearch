# 🧪 分子性质预测

基于图神经网络的分子性质预测自主研究框架。

## 🎯 概述

本子项目实现了使用GNN（图神经网络）从分子图结构预测分子性质的自主研究。

## 📁 文件说明

```
molecular/
├── prepare_mol.py    # 分子图数据生成器
├── train.py          # GNN模型+训练循环（agent修改）
├── program.md        # Agent研究指令
└── README_CN.md      # 本文件
```

## 🔬 技术细节

### 模型架构

**MolecularGNN** - 消息传递神经网络
- **输入**: 分子图（原子特征 + 邻接矩阵）
- **架构**:
  - 节点投影层
  - 3个消息传递层（带残差连接）
  - 图级别池化（平均聚合）
  - 回归输出投影
- **参数量**: ~15K
- **训练**: AdamW优化器，MSE损失

### 数据

**简化分子数据集**
- **训练集**: 5,000个随机分子图
- **验证集**: 500个图
- **特征**:
  - 5种原子类型 (H, C, N, O, F)
  - 每个分子最多29个原子
  - 随机键连接模式
  - 启发式性质目标

## 🤖 自主研究

### 工作原理

1. **AI Agent** 阅读 `program.md` 获取指令
2. **修改** `train.py`（agent唯一可修改的文件）
3. **训练** 严格5分钟
4. **评估** 验证集MSE
5. **保留或丢弃** 基于改进效果决定
6. **重复** 自主迭代

### 改进策略

Agent可以探索：
- **架构**: GNN深度、隐藏维度、激活函数
- **训练**: 学习率、批次大小、优化器类型
- **正则化**: Dropout、权重衰减、批归一化
- **损失函数**: MSE变体、Huber损失
- **高级特性**: 边特征、注意力机制、池化策略

### 启动Agent研究

```bash
# 在AI聊天界面 (Claude/Codex等)

"请阅读 molecular/program.md，然后开始第一个改进实验。
请说明你的假设，进行修改，并运行 train.py 验证效果。"
```

## 📊 设计原则（来自原始autoresearch）

✅ **5分钟时间预算** - 实验可比较
✅ **单文件修改** - 仅修改 `train.py`
✅ **自包含** - 最小依赖
✅ **可审查diffs** - 清晰的改动历史
✅ **单一指标** - MSE优化
